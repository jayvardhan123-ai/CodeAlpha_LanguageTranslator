## 💡 How to Capture High-Quality Screenshots

1. **Maximize Window**: Run the Streamlit application in full-screen on a modern browser.
2. **Crop Correctly**: Only capture the browser viewport (exclude the desktop taskbar, URL bar, or bookmark folder rows).
3. **Optimized Formats**: Save them in `.png` format. For high performance on GitHub, keep the size under 1MB per image (you can compress them using online tools if needed).
4. **Consistency**: Use a dark window style consistently for all screenshots to showcase the high-end neon glows and glass details!
